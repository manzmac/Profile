Button Style

I’amRex has 3 diffrent Button Style and a special class name associated with each of them.

Background Button
  • Normal Button Style .rex-primary-btn
  • Effects Button Style .rex-primary-btn-effect
  • Background Effects Button Style .rex-btn-background-effect
Without Background Button
  • Normal Button Style .rex-primary-btn-No
  • Effects Button Style .rex-primary-btn-effect-No
  • Background Effects Button Style .rex-btn-background-effect-No
